Commands

Hello! Welcome to Iroha Asobe commands page. You can see all her commands in this page or you can use the command !help on discord to get a help menu.

Important information:
The commands will be explained with the following format, first the prefix of the command, then the name and lastly the arguments !help [arguments].
If the argumets are like this <arguments> means that it is a mandatory argument, whereas if the arguments is with this [arguments] means that it is a optional argument.

If you need more help you can join to our bot support server for more explicit and detailed help, our staff will gladly answer all your questions or problems. Click here to join!.

ADMIN/MOD Commands

Name Arguments Description
!kick <target user's @> Kicks a member from your discord server
!ban <target user's @> Bans a member from your discord server
!unban <User ID> Unbans a member from your server
!clear <number from 1-100> Deletes a specified ammount of messages
!suggestion <Suggestion ID> <WAITING,ACCEPTED,DENIED> [Reason] Updates the status of a suggestion

Configuration Commands

Name Arguments Description
!command <"enable" or "disable"> <Command Name> Enables or disables a command for this guild
!language [language] Displays or sets the language for this Discord server
!prefix [new prefix] Displays or sets the prefix for the current guild
!requirerole <Command Name> <"none" | Tagged Role | Role ID String> Specifies what role each command requires.
!setreports Sets the reports channel
!setsuggestions Sets the suggestions channel
